Introduction of .NET :

  1. Introduction :

    .NET is a comprehensive and versatile developer platform created by Microsoft for building a wide range of applications. It supports multiple programming languages, libraries, and frameworks, enabling developers to create web, mobile, desktop, gaming, and Internet of Things (IoT) applications.

    • .NET Framework is a software development framework.

    • It includes a library and runtime environment.

    • FCL provides user interface, database connectivity, cryptography, web development and many more.

    • .NET programs are executed in CLR (Common Language Runtime).

    • Provides lenguage interoperability.

  3. .NET Stack :

    The .NET stack is a collection of tools, frameworks, and libraries used for developing applications on the .NET platform.
    The image showcases the architecture of the .NET Framework, including its various components and their relationships.

    • Core Components :

      • Common Language Runtime (CLR) :

        The foundation of the .NET Framework, responsible for executing .NET applications. It provides services like memory management, garbage collection, and security.

      • Framework Class Library (FCL) :

        A large collection of reusable classes, interfaces, and value types that provide a wide range of functionality for developers. It includes classes for working with strings, files, networking, data access, and much more.

    • Application Development Technologies :

      • Windows Forms (WinForms) :

        A graphical user interface (GUI) framework for creating Windows desktop applications.

      • ASP.NET :

        A web application framework for building dynamic websites and web services.

      • WCF (Windows Communication Foundation) :

        A framework for building service-oriented applications and enabling communication between distributed systems.

      • WPF (Windows Presentation Foundation) :

        A UI framework for building rich, interactive Windows desktop application using XAML.

      • WF (Windows Workflow Foundation) :

        A framework for creating and managing workflows within .NET applications.

    • Data Access Technologies :

      • ADO.NET :

        A set of classes for accessing and manipulating data from various data sources, such as SQL Server, Oracle, and XML.

      • Entity Framework :

        An object-ralational mapping (ORM) framework that simplifies database interactive by allowing developers to work with data in terms of objects.

    • Other Technologies :

      • LINQ (Language Integrated Query) :

        A set of language extensions that allow developers to write queries against various data sources, including collections, databases, and XML documents.

      • Parallel LINQ :

        An extension of LINQ that enables parallel processing of queries to improve performance.

      • Task Parallel Library :

        A set of classes for writing parallel and asynchronous code.

      • Task-Based Async Model :

        A programming model for writing asynchronous code using tasks and the async / await keywords.

      • .NET APIs for Store/UWP apps :

        APIs for building Universal Windows Platform (UWP) apps that can run on Windows 10 devices.

      • Card Space :

        A deprecated technology for managing digital identities.

    Overall, the .NET Framework is a comprehensive platform that provides developers with a wide range of tools and technologies for building diverse applications.

  4. Advantages

    .NET offers a wide range of advantages that make it a popular choice among developers for building diverse types of applications.

    • Object-oriented programming.

    • Language independence.

    • Efficient data access.

    • Code sharing.

    • Support for services.

    Advantages Description
    Cross-Platform Development Supports Windows, macOS, and Linux, allowing you to write once and run anywhere.
    Language Interoperability Works with multiple languages like C#, F#, and Visual Basic, enabling seamless integration.
    High Performance Optimized execution through Just-In-Time (JIT) compilation and efficient memory management.
    Rich Libraries Extensive libraries and frameworks (e.g., ASP.NET Core, Entity Framework Core) simplify development.
    Robust Security Features like Code Access Security (CAS) and role-based security enhance application safety.
    Developer Productivity Advanced IDEs (Visual Studio, VS Code) with features like IntelliSense, debugging, and code refactoring.
    Scalability and Reliability Easy integration with Azure for building scalable, reliable cloud applications.
    Unified Framework A single platform for all types of applications, reducing learning curves.
    Active Community Large, supportive community and comprehensive documentation.
    Modern Development Practices Built-in support for asynchronous programming and dependency injection.
    Regular Updates Continuous evolution with new features, performance improvements, and security enhancements.

    .NET's cross-platform capabilities, language interoperability, high performance, comprehensive libraries, robust security, and strong developer tools make it a powerful platform for building a wide variety of applications. Its consistent updates, community support, and modern development practices further enhance its appeal to developers.

.NET Framework - Compilation Process :

  1. Introduction of Compilation Process :

    The compilation process in the .NET Framework is designed to convert source code written in high-level languages like C# or VB.NET into a form that can be executed by the .NET runtime environment. This process involves several stages, from initial code writing to final execution.

    • Compilation process is done in two parts.

    • First phase is done by language compilers.

    • Second phase is done by JIT.

    • .NET programs are executed in CLR (Common Language Runtime).

    • Provides lenguage interoperability.

    Doctype declaration tells our web server that we are coding in HTML.

    1. HTML 4.01

      <!DOCTYPE HTML PUBLIC "//W3C//DTDHTML4.01Transitional//EN""http://www.w3.org/TR/html4/loose.dtd">

      <!DOCTYPE HTML PUBLIC "//W3C//DTDHTML4.01//EN""http://www.w3.org/TR/html4/strict.dtd">

      <!DOCTYPE html> And <!doctype html>

Typography tags :

Typography tags in HTML are elements used to structure and style text content on web pages, ensuring that text is presented in a clear, readable, and visually appealing manner. These tags help define the hierarchy, emphasis, and semantic meaning of the text. Here are some of the key typography tags in HTML :

  1. Heading tag

  2. Paragraph tag

  3. Centered tag

  4. Moving tag

    5. Text Formatting :
    • <b> : Bold text.
    • <strong> : Important text, typically rendered in bold.
    • <i> : Italic text.
    • <em> : Emphasized text, typically rendered in italics.
    • <mark> : Highlighted text.
    • <small> : Smaller text.
    • <del> : Deleted text (strikethrough).
    • <ins> : Inserted text (underline).
    • <sub> : Subscript text.
    • <sup> : Superscript text.
    6. Quotes and Citations :
    • <blockquote>: Block-level quotation.
    • <q>: Inline quotation.
    • <cite>: Citation or reference.
    • <abbr>: Abbreviation or acronym.
    7. Lists :
    • <ul>: Unordered list.
    • <ol>: Ordered list.
    • <li>: List item.
    8. Preformatted Text :
    • <pre>: Preformatted text, preserving spaces and line breaks.
    9. Code and Keyboard Input :
    • <code>: Inline code.
    • <kbd>: Keyboard input.
    • <samp>: Sample output.
    10. Span and Div :
    • <span>: Inline container for text.
    • <div>: Block-level container for text or other elements.
  1. Heading text tags :

    <h1> to <h6>: Define headings.
    Heading tags in HTML are used to define headings for web content, providing a way to structure and organize information hierarchically. They range from <h1> to <h6>, with <h1> being the most important and <h6> the least important. These tags not only affect the visual presentation of the content but also play a crucial role in SEO and accessibility.
    Main heading : <h1>
    Sub headings : <h2>, <h2>, <h3>, <h4>, <h5> and <h6>

    
                            <h1> heading one </h1>
                            <h2> heading second </h2>
                            <h3> heading third </h3>
                            <h4> heading fourth </h4>
                            <h5> heading fifth </h5>
                            <h6> heading sixth </h6>
  2. Paragraph text tag :

    <p>: Defines a paragraph.
    The <p> tag in HTML is used to define a paragraph of text. It is a block-level element, meaning it will automatically start on a new line and take up the full width available. The <p> tag is one of the most basic and commonly used elements in HTML, helping to structure and format text content.

    
                            <p> Welcome to my html website. </p>
  3. Centered text tag :

    The <center> tag in HTML was used to center-align text and other elements within its container. However, it is important to note that the <center> tag is deprecated in HTML 4.01 and not supported in HTML5. This means it should not be used in modern web development as it is no longer considered a best practice for styling content. Instead, CSS should be used to achieve centering.

    
                            <p> Center content write  </p>
  4. Moving text tag :

    The <marquee> tag in HTML was used to create a scrolling text or image display within a web page. However, it is important to note that the <marquee> tag is obsolete and deprecated in HTML5. This means it should not be used in modern web development as it is no longer supported by HTML standards and is not guaranteed to work in all browsers. Instead, CSS and JavaScript should be used to create scrolling or animated text and images.


      Attributes of the <marquee> Tag :
    • direction : Specifies the direction of the scroll (left, right, up, down).
    • behavior : Specifies the scrolling behavior (scroll, slide, alternate).
    • scrollamount : Specifies the speed of the scrolling (numeric value).
    • scrolldelay : Specifies the delay between each scroll movement (numeric value in milliseconds).
    • loop : Specifies the number of times the marquee will loop.
    • bgcolor : Specifies the background color of the marquee.
    
                            <marquee> Moving Content Write Here </marquee>

HTML Document Formatting and Performatting :

HTML Document Formatting and Preformatting refer to how HTML handles and displays content, particularly text.

  1. HTML Document Formatting : HTML Document Formatting involves how text and elements are visually arranged and styled within an HTML document. This includes the use of various HTML tags to control the appearance and structure of content.

      1. Bold Text (<b> and <strong>) :
      • <b> : Used to make text bold without implying any extra importance.
      • <strong> : Used to emphasize text strongly, typically rendered in bold, and implies that the text is of strong importance.
      2. Italic Text (<i> and <em>) :
      • <i> : Used to italicize text without implying any emphasis.
      • <em> : Used to emphasize text, typically rendered in italics, and implies that the text is important.
      3. Underline Text (<u>) :
      • <u> : Used to underline text, though it is often better to use CSS for underlining text for semantic clarity.
      4. Strikethrough Text (<s> and <del>) :
      • <s> : Used to indicate text that is no longer accurate or relevant.
      • <del> : Represents deleted text, often used in document revisions
      5. Subscript and Superscript (<sub> and <sup>) :
      • <sub> : Used to display subscript text.
      • <sup> : Used to display superscript text.
      6. Inline Code (<code>) :
      • <code> : Used to display inline code snippets.
      7. Keyboard Input (<kbd>) :
      • <kbd> : Used to indicate keyboard input.
      8. Quotation (<q> and <blockquote>) :
      • <q> : Used for inline quotations.
      • <blockquote> : Used for block-level quotations, often indented from the rest of the text.
    9. Hr tag :

      Empty element tag

      
                                    <hr>

  2. Performatting : Preformatting refers to preserving the exact formatting of text as it appears in the HTML code. This is achieved with the <pre> tag, which maintains whitespace, line breaks, and formatting, making it useful for displaying code or other preformatted text.

    • Pre tag : 
      
                                    <pre>
                                        This is preformatted text.
                                            It maintains whitespace and line breaks.
                                    </pre>
