  2. Use of Tags :

    Tags are used to display our contents on webpage and it also defines the place of our contents.
    HTML tags are essential for structuring and presenting content on the web. They provide a way to define different parts of a webpage and their respective roles, enabling browsers to display content correctly and uniformly.

    • Content Structuring

      HTML tags organize the content into logical sections, such as headings, paragraphs, lists, and tables. This helps both users and search engines understand the hierarchy and flow of information.

    • Styling and Presentation

      Tags like <div>, <span>, and <style> allow for CSS integration, enabling developers to style and position elements to create visually appealing web pages.

    • Interactivity

      Tags such as <a>, <button>, <form>, and <input> enable user interaction, allowing users to navigate between pages, submit information, and interact with web applications.

    • Media Embedding

      Tags like <img>, <video>, and <audio> allow for embedding images, videos, and audio files, enriching the multimedia experience on web pages.

    • Accessibility

      Semantic tags such as <header>, <footer>, <article>, and <nav> enhance accessibility by providing additional context to assistive technologies, making the web more inclusive for users with disabilities.

    • SEO (Search Engine Optimization)

      Proper use of tags such as <title>, <meta>, <h1>, and <h2> helps improve search engine ranking by providing clear and relevant information about the content of a web page.

    • Behavior and Scripts

      Tags like <script> enable the inclusion of JavaScript to add dynamic behavior and functionality to web pages.

Html Doctype Declarations

  1. DOCTYPE Declarations :

    Doctype declaration is a comment that tells in which version of Html coding is done.
    Doctype declaration tells our web server that we are coding in HTML.

    Types of Doctype Declaration :

    1. HTML 4.01

      <!DOCTYPE HTML PUBLIC "//W3C//DTDHTML4.01Transitional//EN""http://www.w3.org/TR/html4/loose.dtd">

    2. XHTML

      <!DOCTYPE HTML PUBLIC "//W3C//DTDHTML4.01//EN""http://www.w3.org/TR/html4/strict.dtd">

    3. HTML 5

      <!DOCTYPE html> And <!doctype html>

    Note :

    1. Always you have to do doctype declaration in html 5 version.

    2. If you do the doctype declaration in html5 version then you can take advantage of all the features of HTML 4 and XHTML.

Typography tags :

Typography tags in HTML are elements used to structure and style text content on web pages, ensuring that text is presented in a clear, readable, and visually appealing manner. These tags help define the hierarchy, emphasis, and semantic meaning of the text. Here are some of the key typography tags in HTML :

  1. Heading tag

  2. Paragraph tag

  3. Centered tag

  4. Moving tag

    5. Text Formatting :

    • <b> : Bold text.

    • <strong> : Important text, typically rendered in bold.

    • <i> : Italic text.

    • <em> : Emphasized text, typically rendered in italics.

    • <mark> : Highlighted text.

    • <small> : Smaller text.

    • <del> : Deleted text (strikethrough).

    • <ins> : Inserted text (underline).

    • <sub> : Subscript text.

    • <sup> : Superscript text.

    6. Quotes and Citations :

    • <blockquote>: Block-level quotation.

    • <q>: Inline quotation.

    • <cite>: Citation or reference.

    • <abbr>: Abbreviation or acronym.

    7. Lists :

    • <ul>: Unordered list.

    • <ol>: Ordered list.

    • <li>: List item.

    8. Preformatted Text :

    • <pre>: Preformatted text, preserving spaces and line breaks.

    9. Code and Keyboard Input :

    • <code>: Inline code.

    • <kbd>: Keyboard input.

    • <samp>: Sample output.

    10. Span and Div :

    • <span>: Inline container for text.

    • <div>: Block-level container for text or other elements.

  1. Heading text tags :

    <h1> to <h6>: Define headings.
    Heading tags in HTML are used to define headings for web content, providing a way to structure and organize information hierarchically. They range from <h1> to <h6>, with <h1> being the most important and <h6> the least important. These tags not only affect the visual presentation of the content but also play a crucial role in SEO and accessibility.
    Main heading : <h1>
    Sub headings : <h2>, <h2>, <h3>, <h4>, <h5> and <h6>

    
                            <h1> heading one </h1>
                            <h2> heading second </h2>
                            <h3> heading third </h3>
                            <h4> heading fourth </h4>
                            <h5> heading fifth </h5>
                            <h6> heading sixth </h6>

  2. Paragraph text tag :

    <p>: Defines a paragraph.
    The <p> tag in HTML is used to define a paragraph of text. It is a block-level element, meaning it will automatically start on a new line and take up the full width available. The <p> tag is one of the most basic and commonly used elements in HTML, helping to structure and format text content.

    
                            <p> Welcome to my html website. </p>

  3. Centered text tag :

    The <center> tag in HTML was used to center-align text and other elements within its container. However, it is important to note that the <center> tag is deprecated in HTML 4.01 and not supported in HTML5. This means it should not be used in modern web development as it is no longer considered a best practice for styling content. Instead, CSS should be used to achieve centering.

    
                            <p> Center content write  </p>

  4. Moving text tag :

    The <marquee> tag in HTML was used to create a scrolling text or image display within a web page. However, it is important to note that the <marquee> tag is obsolete and deprecated in HTML5. This means it should not be used in modern web development as it is no longer supported by HTML standards and is not guaranteed to work in all browsers. Instead, CSS and JavaScript should be used to create scrolling or animated text and images.

      Attributes of the <marquee> Tag :

    • direction : Specifies the direction of the scroll (left, right, up, down).

    • behavior : Specifies the scrolling behavior (scroll, slide, alternate).

    • scrollamount : Specifies the speed of the scrolling (numeric value).

    • scrolldelay : Specifies the delay between each scroll movement (numeric value in milliseconds).

    • loop : Specifies the number of times the marquee will loop.

    • bgcolor : Specifies the background color of the marquee.

    
                            <marquee> Moving Content Write Here </marquee>

HTML Document Formatting and Performatting :

HTML Document Formatting and Preformatting refer to how HTML handles and displays content, particularly text.

  1. HTML Document Formatting :

    HTML Document Formatting involves how text and elements are visually arranged and styled within an HTML document. This includes the use of various HTML tags to control the appearance and structure of content.

      1. Bold Text (<b> and <strong>) :

      • <b> : Used to make text bold without implying any extra importance.

      • <strong> : Used to emphasize text strongly, typically rendered in bold, and implies that the text is of strong importance.

      2. Italic Text (<i> and <em>) :

      • <i> : Used to italicize text without implying any emphasis.

      • <em> : Used to emphasize text, typically rendered in italics, and implies that the text is important.

      3. Underline Text (<u>) :

      • <u> : Used to underline text, though it is often better to use CSS for underlining text for semantic clarity.

      4. Strikethrough Text (<s> and <del>) :

      • <s> : Used to indicate text that is no longer accurate or relevant.

      • <del> : Represents deleted text, often used in document revisions

      5. Subscript and Superscript (<sub> and <sup>) :

      • <sub> : Used to display subscript text.

      • <sup> : Used to display superscript text.

      6. Inline Code (<code>) :

      • <code> : Used to display inline code snippets.

      7. Keyboard Input (<kbd>) :

      • <kbd> : Used to indicate keyboard input.

      8. Quotation (<q> and <blockquote>) :

      • <q> : Used for inline quotations.

      • <blockquote> : Used for block-level quotations, often indented from the rest of the text.

    9. Hr tag :

      Empty element tag

      
                                    <hr>

  2. Performatting :

    refers to preserving the exact formatting of text as it appears in the HTML code. This is achieved with the <pre> tag, which maintains whitespace, line breaks, and formatting, making it useful for displaying code or other preformatted text.
